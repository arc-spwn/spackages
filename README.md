A repo for packages written in SPWN

# Installation

## Ash
- Go to the directory where Ash is located
- Place desired package in `commands` directory
- Repeat step 1
- Open `commands.spwn`
- Locate the `files` array
- Add the package file there(e.g. `name.spwn`)
